# Naebaecam_Dart_Simhwa

## 다트 심화 과제 - 성적 분석 프로그램

1. 필수 과제 기능
    - 학생 이름 입력 시 이름과 성적을 result.txt에 저장.

2. 심화 과제 기능
    - 가장 점수가 높은 학생의 이름과 성적 출력 & result.txt에 저장.
    - 전체 학생들의 평균 성적 출력.
    - 모든 학생들의 이름과 성적 출력.
    - 학생 이름 입력 시 점수 출력 & result.txt에 저장.
    - 학생 점수 수정.
    - 학생 이름과 성적 추가.
    - 학생 이름과 성적 삭제.
